BCH Z-Score Calculator API

REST API wrapper for the Boston Children's Hospital Z-Score Calculator with dynamic payload support

API Documentation

Endpoint
POST /calculate-zscore
Request Format

Send a JSON payload with the required form fields:

{
  "LastRegressionNum": "12",
  "HeightValue": "172",
  "WeightValue": "32",
  "AgeValue": "1",
  "Context": "Echo",
  "Group": "Aorta",
  "RegressionNum": "12",
  "XAxisValue": "1.11",
  "YAxisValue": "1.2"
}
Response Format
{
  "status": "success",
  "data": {
    "ZScore": "-2.5",
    "Percentile": "0.6",
    "5th": "0.8",
    "50th": "1.2",
    "95th": "1.6"
  },
  "message": "Z-score calculated successfully"
}
